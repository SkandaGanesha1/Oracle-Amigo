# AMIGO SVG path logo animation

This is a React + framer-motion version of the AMIGO path-loader animation.

## Run

```bash
npm install
npm start
```

## Files changed

- `src/App.js` contains the AMIGO SVG paths and the repeated pathLength/pathOffset loader animation.
- `src/styles.css` sets the black page background and white logo strokes.

## Preview exports

The `assets` folder contains a static PNG/SVG and an animated GIF/SVG preview.
